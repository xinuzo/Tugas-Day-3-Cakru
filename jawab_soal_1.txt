Axios adalah sebuah library HTTP client yang ditulis dalam JavaScript untuk menjalankan permintaan HTTP dari browser atau Node.js. Library ini memungkinkan untuk membuat permintaan HTTP dengan mudah, baik itu GET, POST, PUT, DELETE, dan sebagainya. Axios juga mendukung fitur-fitur seperti transformasi data, pembatalan permintaan, dan interceptor. Ini membuatnya menjadi pilihan yang populer dalam pengembangan aplikasi web modern.
